# nextjs-elements-netlify-serverless

## Install

```bash
yarn
```

## Setup env

```bash
cp .env.example .env
```

Add your Stripe keys

## Run

```bash
yarn dev
```

## Test Payment Request Button

See requirements in [docs](https://stripe.com/docs/stripe-js/elements/payment-request-button#react-prerequisites)

```bash
ngrok http 3000
```
